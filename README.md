# CODEWARS

Uma dos métodos que utilizo para exercitar meus conhecimentos em Java é a plataforma Codewars.

## ESTRUTURA

No diretorio src estão os codigos com as soluções que desenvolvi para os desafios.

Em cada classe existe o enunciado do desafio criado pela plataforma, com o problema que deve ser analisado.

No diretorio test é possivel executar o código da solução encontrada para cada desafio.
